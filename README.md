## Publish via GitHub Pages

1. Create a new repository, for example `thomas-klein-cv`
2. Upload all files from this folder
3. Open **Settings → Pages**
4. Set source to **Deploy from branch**
5. Choose **main** branch and `/ (root)`
6. Save and wait until GitHub publishes the site

Your site will then be available under a URL like:

`https://darkangel3010.github.io/thomas-klein-cv/`
